## じゃんけんゲーム
### プロジェクト概要
このプロジェクトは、コンピューターとじゃんけんをして勝敗を楽しむシンプルなゲームです。
Python版とJava版があります。
プログラムを実行すると、コンピューターがランダムに手を出し、集配を判定します。
## 実行環境
- Python版：Python 3.x
- Java版：Java 11以上推奨
## 実行方法
### Python版：janken.py
1. ターミナルでフォルダに移動
2. 以下を実行
```言語名
python janken.py
```
### Java版：Janken.java
1. ターミナルでフォルダに移動
2. コンパイル
```言語名
    javac janken.java
```
3. 実行
```言語名
    janken.Janken
```
### 遊び方
1. プログラムを実行すると、手の選択肢が表示されます。
2. 数字で手を入力します:
    0→グー
    1→チョキ 
    2→パー
3. コンピューターがランダムに手を出し、勝敗が判定されます。
4. 結果が表示されたら、再度挑戦することも可能です。
